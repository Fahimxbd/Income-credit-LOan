# Loan Eligibility Checker (Python)

## Overview
This project is a simple Python-based Loan Eligibility Checker. 
It determines whether a user is eligible for a loan based on their annual income and credit score.

This project is designed for beginners who are learning conditional logic in Python.

## Features
- Takes annual income as input
- Takes credit score as input
- Checks eligibility automatically
- Displays loan eligibility result instantly

## How It Works
The program checks two conditions:

1. Annual income must be greater than 1,000,000
2. Credit score must be greater than or equal to 1000

If both conditions are satisfied, the user is eligible for a loan.

Otherwise, the user is not eligible.

## Requirements
Python 3.x

## How to Run the Program
Run the following command:

python credit.py

## Example Output
